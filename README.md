# 🍸 Bar Atlas ("酔いの地図帳")

## 🌟 Overview
Bar Explorer is a web application designed to help users efficiently find cocktail and whiskey bars on an interactive map. Unlike existing services, which often lack refined filtering options and clear information at a glance, this project aims to create an intuitive and informative experience for bar enthusiasts.

## 🚧 Project Status
This project is currently in the early stages of development.  
Many features described in this README are still under development and have not been pushed to this repository yet.

## 🍾 Key Features
- **Interactive Map Search**: Find bars based on location, using Google Maps integration.
- **Detailed Bar Information**: Includes name, address, budget range, seating types, smoking policy, and drink specialties.
- **Advanced Filtering**: Search by drink type, budget, seating arrangements, and operating hours.
- **Favorites & Visit Logs**: Users can save favorite bars and log their visits with notes.
- **Mobile-Friendly UI**: Optimized for both desktop and mobile use.

## 🚀 Tech Stack
- **Backend**: Ruby on Rails 7.2.2.1, PostgreSQL
- **Frontend**: Tailwind CSS, Stimulus.js
- **Maps API**: Google Maps / Places API
- **Authentication**: Devise (planned)

## 🛠 Installation (WIP)
```bash
# Clone the repository
git clone https://github.com/yourusername/bar-explorer.git
cd bar-explorer

# Install dependencies
bundle install

# Set up the database
rails db:create db:migrate db:seed

# Start the development server
rails server
```
